DEBRIEF PEDAGÓGICO — V18 BACKEND INTEGRADO

1. Publique toda esta pasta no GitHub Pages.
2. Abra facilitadora.html numa janela privada/não partilhada.
3. Selecione a sessão e introduza a Chave Facilitadora correspondente.
4. Envie aos docentes apenas a ligação exibida na consola: participar.html?sessao=...
5. Use os botões da consola para abrir cada atividade. Os telemóveis atualizam automaticamente.
6. Os resultados aparecem na mesma consola e são atualizados a cada 5 segundos.

Segurança:
O URL do Power Automate está no ficheiro config_backend.js. Não partilhe o ZIP publicamente fora do repositório necessário. Depois das sessões, recomenda-se regenerar o URL/assinatura do acionador HTTP, porque o endereço funciona como uma chave técnica.
