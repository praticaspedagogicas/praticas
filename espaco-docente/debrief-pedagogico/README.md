# Debrief Pedagógico — Práticas em Balanço · V16

## Forms já integrados

As quatro ligações fornecidas pela utilizadora já estão incorporadas no `index.html` e nos QR codes locais da pasta `assets`. Não é necessário voltar a colá-las.

## Ficheiros principais

- `index.html` — preparação e ecrã projetado.
- `facilitadora.html` — consola privada.
- `ROTEIRO_FACILITADORA_PASSO_A_PASSO.txt` — instruções operacionais.
- `FORMS_INTEGRADOS_E_RENOVACAO.txt` — ligações e ciclo entre sessões.
- `forms_kit/` — documentos e instruções de apoio já utilizados para criar os Forms.

## Três sessões distintas

No início, selecione **Universidade Europeia**, **IADE** ou **IPAM**. Depois de cada sessão, exporte e elimine as respostas dos quatro Forms. No painel, use **Preparar próxima sessão** para limpar votos, notas e cronómetros locais, preservando as ligações.

> O HTML não consegue apagar respostas da conta Microsoft. A confirmação da limpeza é obrigatória no fluxo de nova sessão, mas a ação deve ser feita no próprio Microsoft Forms.

## Publicação

Publique a pasta completa no GitHub Pages, preservando `assets/` e `forms_kit/`. Não publique apenas o HTML isolado.
